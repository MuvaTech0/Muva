AQUÍ PODRAS REALIZAR:

Examen Front-End Técnico

El alumno deberá manejar y representar visualmente los datos obtenidos de una API externa bajo ciertas condiciones dadas por nosotros.

Contenido a evaluar:

  • -Codigo limpio, legible y comentado.
  • -Implementación de conceptos de la propia tecnología.
  • -Implementacion de una libreria CSS especifica.
  • -Generación de peticiones a API externa.
  • -Manejo de respuestas JSON vía fetch.

Precio:

  • Alumnos de Muva : GRATIS
  • Autodidactas : $1000 ARS

Certificado: Front End Developer

Duración : 1:30 hs

Examen Back-End Técnico

El alumno deberá construir la API la logica de negocio necesaria para responder las peticiones.

Contenido a evaluar:

  • -Creación de CRUD.
  • -Implementación de Middlewares.
  • -Creación e implementacion de controladores.
  • -Generación de endpoints para su llamada a petición.

Precio:

  • Alumnos de Muva : GRATIS
  • Autodidactas : $1000 ARS

Certificado: Back End Developer

Duración : 1:30 hs.

Examen Resolución de problemas I

El alumno deberá solucionar y reconstruir codigo dado por nosotros, llegando al objetivo final planteado por nosotros.

Contenido a evaluar:

  • -Resolución de diferentes problemas técnicos planteados.
  • -Entendimiento de codigo construido en INGLÉS por parte de otros desarrolladores.

Precio:

  • Alumnos de Muva : GRATIS
  • Autodidactas : $1000 ARS

Certificado: Resolución de Problemas I

Duración : 1 h
